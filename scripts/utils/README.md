# Utility Scripts

通用工具脚本目录。

## 状态

⬜ **规划中**（v0.3+）

## 计划功能

此目录计划存放通用工具脚本，供其他模块调用：

- 备份工具（配置文件备份、恢复、清理过期备份）
- 回滚工具（统一的回滚管理）
- 检查工具（安全状态检查、合规性检查）
- 报告工具（已移至 `scripts/base/report.sh`）

## 目录结构（规划）

```
utils/
├── backup.sh       # 备份管理工具
├── rollback.sh     # 统一回滚管理
└── check.sh        # 安全状态检查工具
```

## 注意

基础工具函数已在 `scripts/base/utils.sh` 中实现，此目录用于更高级的工具脚本。
