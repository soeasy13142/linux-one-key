# Server Software

服务器软件安装脚本目录。

## 状态

⬜ **规划中**（v0.3+）

## 计划功能

此目录计划存放常用服务器软件的安装和配置脚本：

- Web 服务器（Nginx, Apache）
- 数据库（MySQL, PostgreSQL, Redis）
- 缓存（Redis, Memcached）
- 消息队列（RabbitMQ）
- 监控（Prometheus, Grafana）

## 目录结构（规划）

```
server/
├── nginx.sh        # Nginx 安装与配置
├── mysql.sh        # MySQL 安装与安全配置
├── redis.sh        # Redis 安装与配置
└── postgresql.sh   # PostgreSQL 安装与配置
```
