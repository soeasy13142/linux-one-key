# Dev Tools

开发工具安装脚本目录。

## 状态

⬜ **规划中**（v0.3+）

## 计划功能

此目录计划存放开发环境相关的安装脚本：

- 编辑器/IDE 配置（vim, nano）
- 版本控制工具（git 配置）
- 编译工具链（gcc, make, cmake）
- 运行时环境（Node.js, Python, Go）
- 容器工具（Docker, containerd）

## 目录结构（规划）

```
dev/
├── editor.sh       # 编辑器配置
├── git.sh          # Git 配置
├── docker.sh       # Docker 安装
└── runtimes.sh     # 编程语言运行时
```
